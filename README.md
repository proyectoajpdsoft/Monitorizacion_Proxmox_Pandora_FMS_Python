Monitorizar sistema de virtualización Proxmox con Pandora FMS

Monitorizar un sistema de virtualización Proxmox mediante Pandora FMS. Monitorear estado del clúster Proxmox, de los OSD del almacenamiento hiperconvergente Ceph, CPU, RAM, procesos, cron, número de máquinas virtuales y contenedores, servicios Ceph, PVE.
Más información en: https://proyectoa.com/monitorizar-sistema-de-virtualizacion-proxmox-con-pandora-fms/

\## Características principales

\- Fichero de configuración de PandoraFMS Agent (pandorafms_agent.conf) con la ejecución del script Python proxmox.py

\- Script Python proxmox.py que obtiene todos los datos del estado de un clúster Proxmox y Ceph.

\- Fichero XML de ejemplo de salida de ejecución de proxmox.py


\## Tecnologías utilizadas

\- Python 3.9+

\- Proxmox 8.4

\- PandoraFMS Community
